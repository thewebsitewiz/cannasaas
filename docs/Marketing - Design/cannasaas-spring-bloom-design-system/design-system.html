Tailwind Config

Drop-in config for all HTML pages. Extends default theme with the full palette + font families.

    extend: {
      colors: {
        brand: {
          50:  '#F2F9EE', 100: '#E0F1D5', 200: '#C3E4AE',
          300: '#9BD37D', 400: '#73BF52', 500: '#56A638',
